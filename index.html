PROJECTS

Algo Rhythm

A single page application allowing the user to play audio files using interactive scroll bars and volume controls. React powers the frontend layout and interactivity, and React Router enables state-based routing. Built as part of the Bloc Web Development curriculum.

RBG Color Picker

The user is given an RGB color value and must select one of three colors that matches this value. "Easy" and "hard" modes present the user with 3 and 6 possible color choices, respectively. Built with plain JS and styled with CSS, this simple app packs a lot of functionality into not a lot of code. Based on a Colt Steel Web Development Bootcamp project.
